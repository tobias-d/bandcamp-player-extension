# Welcome Overlay (First Launch)

## Headline
**Welcome to Bandcamp Player BPM Analyzer**

Thanks for downloading and installing the extension.

## Intro Copy
This quick overlay appears only the first time you open the plugin.
It covers the full panel area below the top controls so you can get started in seconds.

## Main Goals
- **Accurate BPM detection** for faster, more reliable track analysis.
- **More fun, intuitive digging** so discovering music feels smoother and more enjoyable.

## Feedback Prompt
If you notice inaccurate BPM readings, please send feedback using the **i** button at the top.
Your report helps improve analysis quality for everyone.

## Usability Tip
You can resize the panel to your preferred size by dragging from **any edge**.

## Primary Action
**Let’s go**

Use a large full-width button at the bottom of the overlay:
- Label: **Let’s go**
- Style: high-contrast, prominent, easy to tap/click
- Behavior: close this welcome panel and never show it again
